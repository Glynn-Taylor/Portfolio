Code Cracker (formerly Such Puzzle)

About

Code Cracker is a 3D visual puzzle game created for a 3 week Warwick Game Design competition. It involves searching individual rooms to find a code to open the door to the next room. It was inspired by a similar kongregate game and was created based on the theme: "Monotone".

Features

  • Simple visual puzzles and tricks
  • 12 seperate puzzles to solve
  • Minimilistic graphical style
  • Keypad interaction to get to new rooms

Technical aspect

Made in Unity3D, models created using blender and animated within Unity.
