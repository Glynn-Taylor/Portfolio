Point Cloud Renderer

About

As part of my part time work for Warwick Manufacturing group I worked on a project that took large point clouds data files and converted them into traversable meshes. The project took around 8 weeks and was used in the recreation of Wells cathedral (partially shown left), and a building site, in Unity3D.

My contribution to the project was writing code to convert the point clouds, as well as several tools to place and modify models and interactable objects within the scene at run-time, in order to allow a user to quickly set up a showcase or demo of a historic environment.

Features

  • Automatic meshing and poly reduction of commercial size point clouds
  • Texturing using associated HDR data
  • Tools to import clouds at runtime as well as other objects to quickly create a scene

Technical aspect

Made in Unity3D, point cloud data supplied by Warwick Manufacturing Group.
